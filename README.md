# NovaStore E-Ticaret Veri Yönetim Sistemi

Bu proje, SQL Server üzerinde ilişkisel veri tabanı tasarımı, tablo oluşturma, örnek veri girişi ve raporlama sorgularını uygulamak amacıyla hazırlanmıştır.

## Proje İçeriği

- `Serdarural_NovaStore_Proje.sql`: Veritabanı oluşturma, tablolar, ilişkiler, örnek veriler, sorgular, view ve backup komutları.
- `Serdarural_NovaStore_Proje.docx`: Proje raporu, kod açıklamaları ve örnek sonuçlar.
- `Serdarural_NovaStore_Proje.png`: Veritabanı tablolarının ilişkisel şeması.

## Veritabanı Yapısı

Projede aşağıdaki tablolar bulunmaktadır:

- Categories
- Products
- Customers
- Orders
- OrderDetails

## Kullanılan SQL Konuları

- CREATE DATABASE / CREATE TABLE
- Primary Key / Foreign Key
- IDENTITY, DEFAULT, UNIQUE
- INSERT INTO
- SELECT, WHERE, ORDER BY
- INNER JOIN / LEFT JOIN
- GROUP BY, COUNT, SUM
- Subquery
- DATEDIFF ve GETDATE
- CREATE VIEW
- BACKUP DATABASE

## Çalıştırma

1. SQL Server Management Studio'yu açın.
2. `Serdarural_NovaStore_Proje.sql` dosyasını açın.
3. Scripti sırayla çalıştırın.
4. Sonuç sorgularını çalıştırarak raporları görüntüleyin.

> Yedekleme komutundaki `C:\Yedek\` klasörünün SQL Server tarafından erişilebilir olması gerekir.
